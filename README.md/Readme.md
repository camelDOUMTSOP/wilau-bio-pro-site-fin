# Wilau Bio - Premium Cosmetic Website

Ce projet est une maquette intégrée pour "Wilau Bio", une marque de cosmétiques haut de gamme destinée aux peaux africaines. Le design est luxueux, élégant, et met en avant l'expertise de la fondatrice en esthétique et cosmétologie.

## Structure du Projet

```text
wilau-bio/
│
├── index.html               # Page d'accueil (Hero, Présentation, Produits, Témoignages)
├── produits-services.html   # Grille de produits & présentation du cabinet esthétique
├── a-propos-contact.html    # Histoire de la marque, valeurs, expertise et formulaire de contact
├── blog.html                # Journal beauté avec aperçus d'articles
│
├── css/
│   ├── style.css            # Styles principaux (Typographie, Couleurs, Composants)
│   └── responsive.css       # Media queries pour mobile et tablette
│
├── js/
│   └── main.js              # Menu mobile, Sticky Header, et animations au défilement (Scroll Reveal)
│
└── README.md                # Documentation du projet
